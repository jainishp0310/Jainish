# **App Name**: Eternal Hearts

## Core Features:

- Heart Creation: Create a new heart element on each mouse movement.
- Mouse Tracking: Track mouse position using the 'mousemove' event listener.
- Heart Positioning: Set the heart's position dynamically based on mouse coordinates.
- CSS Animation: Apply fade-out animation to each heart using CSS.
- Dynamic Styling: Apply a random size and rotation to each heart.
- Heart Removal: Automatically remove hearts from the DOM after they fade out.
- AI Name Insertion: The generative AI will optionally use the word 'hetanshi', deciding based on prompt conditions.

## Style Guidelines:

- Primary color: Crimson (#DC143C) to represent passion and love.
- Background color: Off-white (#F0F0F0), providing a soft, clean backdrop.
- Accent color: Pale pink (#FFB6C1), a delicate accent complementing the crimson.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Fade-out animation: Hearts gently fade out after appearing, creating a transient effect.
